Detailed Description

This class encapsulates a stdio FILE (not std::fstream for less overhead), along with associated operations, for easier use in main program.

Notes:

  1. no copies are permitted (copy constructor disabled) since this is meant as one data source per file.
  2. the output file is opened for writing, i.e. the file is replaced. TODO: implement resume function

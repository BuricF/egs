Detailed Description

Dynamics for the 3-component hypercyle extension of the Gray-Scott model (reaction-diffusion equations) 

U + 2 V1  -r[0][0]->  3V1
U + 2 V2  -r[1][1]->  3V2
U + 2 V3  -r[2][2]->  3V3

U + V1 + V2 -r21-> 2V1 + V2
U + V1 + V2 -r12-> 2V2 + V1

U + V1 + V3 -r31-> 2V1 + V3
U + V1 + V3 -r13-> 2V3 + V1

U + V2 + V3 -r32-> 2V2 + V3
U + V2 + V3 -r23-> 2V3 + V2

V1 -k1-> P
V2 -k2-> P
V3 -k3-> P

There are two separate versions of the reaction-diffusion functions, with and witout noise, respectively, as an optimisation. Uses 5-point stencil Laplacian discretisation (for speed)

While technically a concentration operator, the initial conditions loading procedure was included here so that this file would describe the initial value problem.

Detailed Description

Extended Gray-Scott Model Evlution Simulator

This version is built upon the simple simulator, running the dyanmic for a specified length of time.

The simulation features subsequent mutation events until all species go extinct or the alloted times runs out. The roles of wild type and mutant alternate between , , in a deterministic way (see comments and formal algorithm outlined in the thesis report).

This version only features deterministic dynamics beyond the initial conditions randomisation but can easily be adapted following the code in the simple simulator.
